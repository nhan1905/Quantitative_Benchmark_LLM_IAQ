# 2. Requirement for successfully re-implementing our mansucript

  2) Create an API by following these steps: **Settings / API Key / Create API Key** 
- In this study, we used Google Colab platform. Several LLMs takes hours to complete generate solution for the entire 480 problems. Therefore, we encourge to user to use Pro+ version.
- Alternative, other platform can be used to run LLMs on local machine but this will be memory-consuming.
